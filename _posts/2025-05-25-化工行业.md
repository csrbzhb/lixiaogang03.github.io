## 乙烷

乙烷（化学式：C₂H₆）是一种无色、无臭、可燃的气体，属于烷烃类化合物，是最简单的二碳烷烃。它是天然气和石油中的重要成分之一，在石化工业中用途广泛

通过**热裂解（steam cracking）**乙烷可以生成乙烯（C₂H₄），这是制造塑料（如聚乙烯）、溶剂、涂料等的重要原料

### 基础化工原料的主要来源表

| 原料名称     | 来源类型           | 具体来源路径说明                                           |
|--------------|--------------------|------------------------------------------------------------|
| 乙烯         | 石油裂解           | 石脑油裂解、乙烷裂解                                       |
| 丙烯         | 石油裂解           | 与乙烯共裂解、副产自炼油重整                               |
| 苯           | 石油裂解、炼油     | 炼油催化重整、石脑油裂解后萃取                             |
| 甲醇         | 煤化工、天然气合成 | 煤气化 + 合成气（CO+H₂）；天然气蒸汽转化                     |
| 二甲苯       | 炼油重整、芳构化   | 与苯同源，副产于芳烃抽提                                  |
| 苯乙烯       | 加成反应合成       | 苯 + 乙烯 → 苯乙烯（经乙苯）                              |
| PTA          | 芳烃氧化           | 二甲苯（PX）氧化生成                                      |
| 乙二醇       | 环氧乙烷水解       | 乙烯 → 环氧乙烷 → 乙二醇                                   |
| 氨           | 空气分离 + 合成气   | 空气中氮气 + 合成气中氢气（哈柏法）                         |
| 硫酸         | 硫矿焙烧、烟气脱硫 | 焚烧硫铁矿/硫磺，或冶炼废气回收 SO₂                        |
| 盐酸         | 副产或合成         | 多为氯碱工业副产，也可由氢气与氯气直接合成                  |
| 氢氧化钠（烧碱） | 电解工业盐水       | 与氯气共产（电解 NaCl 溶液）                               |
| 氯气         | 电解工业盐水       | 与烧碱共产                                                 |
| 氧气、氮气   | 空气分离           | 深冷分离空气（空分装置）                                   |
| 碳酸钠（纯碱） | 天然矿、人工合成    | 天然碱矿开采或通过“索尔维法”用盐和石灰合成                  |
| 乙炔         | 电石法、天然气裂解 | 电石（CaC₂）加水生成；也可由天然气部分氧化制得              |
